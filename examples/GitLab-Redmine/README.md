# GitLab + Redmine

Start
```bash
cd .efig
sudo ./efig.sh
```
Open URLs in your browser

Resourse | URL
---------|-----
[GitLab](https://github.com/sameersbn/docker-gitlab) | http://gitlab.doc/
[Redmine](https://github.com/sameersbn/docker-redmine) | http://redmine.gitlab.doc/

Stop
```bash
cd .efig
sudo ./efig.sh rm
```
