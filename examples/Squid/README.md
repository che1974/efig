# Squid

Caching all HTTP-data fro all containers

See more information on [jpetazzo/squid-in-a-can](https://github.com/jpetazzo/squid-in-a-can)

Change path to cache folder on efig.yml

```bash
# start container
cd .efig
sudo ./efig.sh
```
```bash
# stop container
cd .efig
sudo ./efig.sh rm
```
